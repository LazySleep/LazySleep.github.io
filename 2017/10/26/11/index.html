安装Android Studio 2.3.3

刚开始学习安卓开发，在eclipse和Android Studio之间选择了后者
总瞎折腾搞得经常要重新安装，留下笔记备忘。
用的是系统是win10

1、Jdk下载。安装。配置#

Android Studio最新版必须使用 Jdk 1.8
http://www.oracle.com/technetwork/java/javase/downloads/index.html

安装时要记得安装位置配置的时候需要用到
右键此电脑—属性—高级系统设置—高级—环境变量
在系统变量中设置

JAVA_HOME：
新建JAVA_HOME，添加JDK根目录的路径

Path：
找到Path，选中后编辑。新建 %JAVA_HOME%\bin（这里与win7不同）
win7下是在开头添加 %JAVA_HOME%\bin；
这样的好处就是更改jdk位置的时候只要改JAVA_HOME就好了


CLASSPATH；
新建 CLASSPATH，添加 .;%JAVA_HOME%\lib 如图

2、Android Studio 下载 安装#

Android Studio 中文社区:
http://www.android-studio.org/
选择对应操作系统，这里选包含SDK的

下载后安装一路next，到这里可以这样勾选，第三个是安卓虚拟机，需要在电脑上调试可以勾选

一路next，这里分别是Android Studio的安装位置和SDK的安装位置

接着还是一路Next就可以完成安装，打开Android Studio
之前没有使用过Android Studio，选择第二个

等待过后出现这个提示，选择Setup Proxy


然后OK或者Next向下，这里选择第二个自定义

选择风格

改为本地sdk路径

点击Finish
安装完成
如果需要汉化的，可以在中文网中找到汉化，有详细使用介绍

3、第一次启动Android Studio#

Android-studio第一次开始跳过SDK下载
在安装目录下/bin下，找到idea.properties文件，以文本方式打开，在末尾添加一行disable.android.first.run=true即可。
再次打开Android-studio，可以直接跳至开始页。

初次安装或者升级Android Studio之后，新建项目，很多人会停在如下图所示的界面上，一直不动。

这是Android Studio在下载Gradle，由于有墙，下载速度可能就几Kb，就会造成卡顿现象。

手动下载Gradle

找到位于 系统盘:\Users\用户名.gradle\wrapper\dists的目录，其中gradle-x.xx-xxx-all就是所有使用过的Gradle版本。
根据gradle-x.xx-xxx-all显式的版本号，去Gradle Build Tools手动下载相对应的complete版本。
https://gradle.org/releases/

下载完后，无需解压，直接放入相对应版本的
类似于55gk2rcmfc6p2dg9u9ohc3hw9样式的目录下

再启动Android Studio会自动解压。
重新打开Android Studio之后，还会有类似于gradle building project info之类的出现，此时是在下载相对应的依赖。

其实接下来已经可以正常使用了，但是我在使用过程中出现了一些有关jdk的问题，网上查了一下要设置本地jdk

文件–其他设置–Default Project Structure

取消勾选，设置本地jdk路径
